# Smart Parking

* ## Bibliothèque nécessaire à la librairie graphique SFML sous Linux
    * freetype
    * x11
    * xrandr
    * udev
    * opengl
    * flac
    * ogg
    * vorbis
    * vorbisenc
    * vorbisfile
    * openal
    * pthread


* ## Comment générer le Makefile avec Cmake : 

    * `cd [CHEMIN-VERS-REPO]/smart-city`
    * `cmake -B build`

* ## Comment compiler avec le Makefile:
    * `make -C build`

* ## Comment executer :
    * `./build/src/smart-city_exec`
